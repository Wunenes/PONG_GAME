Pong Game (Python Turtle)
This is a simple Pong Game created using Python's turtle module. The game features two paddles and a ball that bounces between them, following the classic Pong mechanics.

How to Play
Player 1 (Left Paddle)

Move Up: W
Move Down: S
Player 2 (Right Paddle)

Move Up: Up Arrow
Move Down: Down Arrow
The ball moves automatically, and players must prevent it from passing their paddle.

The game resets when a player misses the ball.
